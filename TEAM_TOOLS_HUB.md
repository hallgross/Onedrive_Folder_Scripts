## ▶️ YouTube / vidIQ — Video Publishing Process
*Run this checklist for EVERY HHD video. Don't publish until all boxes are ✓ (ties into the [LAUNCH READY](./LAUNCH_READY.md) gate: Video + SEO).*
Per-video tracking sheet: [HHD_YouTube_Checklist.csv](./HHD_YouTube_Checklist.csv)

### 1. 🔎 Keyword Research (vidIQ)
- [ ] Pull keywords in vidIQ for the topic (faith, prayer, hope, worship, healing, encouragement, testimony)
- [ ] Pick **1 primary** + **3–5 secondary** keywords (good volume, low–medium competition)
- [ ] Confirm search intent matches the message (someone hurting/seeking would find it)
- [ ] Save chosen keywords in the video's row

### 2. 🏷️ SEO Checklist
- [ ] **Title:** primary keyword near the front · emotional/benefit hook · ≤ 60 chars (e.g. *"Faith Over Fear: A Prayer When You Feel Like Giving Up"*)
- [ ] **Description:** first 2 lines carry the hook + primary keyword; full description 150+ words with secondary keywords woven in naturally
- [ ] **Scripture** quoted in description (anchor verse)
- [ ] **Links** in description: store product · prayer page · related WOW chapter
- [ ] **Tags:** primary + secondary keywords + brand tags (Heavenly Hands Divine, Faith Over Fear)
- [ ] **Hashtags:** 3 in description (#FaithOverFear #Prayer #Worship)
- [ ] vidIQ SEO score checked & improved

### 3. 🖼️ Thumbnail Checklist
- [ ] Brand palette (cream · gold · navy · water tones)
- [ ] Loretta's face/expression OR a clear emotive image (when appropriate)
- [ ] **≤ 4 words** of large, legible text
- [ ] High contrast; readable at phone size
- [ ] Consistent HHD style across the channel
- [ ] A/B test in vidIQ if available

### 4. ⬆️ Upload Checklist
- [ ] Final video file (correct aspect/resolution)
- [ ] Title + description + tags pasted
- [ ] Category set · captions/subtitles uploaded
- [ ] Visibility = scheduled to the launch date
- [ ] Made-for-kids set correctly · comments on (moderated)

### 5. 🔚 End Screen Checklist
- [ ] Subscribe element
- [ ] Next video / best-for-viewer element
- [ ] Playlist element
- [ ] Product or prayer link element

### 6. 🗂️ Playlist Assignment
- [ ] Add to the right playlist(s): **WOW Teaching** · **Prayer** · **Worship** · **Testimony** · monthly-collection playlist

### 7. 🛍️ Product Links
- [ ] Link the matching store product (from the WOW product family / EDU / Juneteenth)
- [ ] Use the **owned/permanent** link (from `HHD_ASSET_REGISTRY.csv`), not a temporary one

### 8. 📖 Scripture References
- [ ] Anchor scripture chosen, on-screen and in description
- [ ] Matches the chapter/product's master scripture

### 9. 🙏 Prayer Links
- [ ] Link to the prayer page / prayer scroll for the topic
- [ ] Clear call to prayer in the video and description

## How this connects to the rest of the system
- **Launch Ready gate:** completing this checklist flips a product's **Video ✓** and **SEO ✓** boxes in `HHD_Launch_Readiness.csv`.
- **One source of truth:** product/scripture/prayer links come from the master records (WOW publishing workflow), never re-typed.
- **Monthly calendar:** each month's collection gets its own playlist + email + social, all gated before going live.
